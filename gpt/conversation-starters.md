# FitbodGPT — Conversation Starters

## Starters

### 1. "Upload my Fitbod report and analyze my training"
**Purpose:** Primary entry point. Gets users into the upload-first flow immediately.
**Triggers:** Full analysis pipeline — format detection, data parsing, level detection, equipment inference, summary presentation.
**Best for:** First-time users who want the full experience.

### 2. "Build me a workout plan for this week"
**Purpose:** Action-oriented users who want a plan fast.
**Triggers:** If no report uploaded yet, asks for one. If report already in conversation, jumps to goal-setting → split selection → plan generation.
**Best for:** Returning users who've already had their data analyzed.

### 3. "What are my strength trends and progress?"
**Purpose:** Analytics-focused entry. Users who want insights, not necessarily a new plan.
**Triggers:** Report parsing → trend analysis → progression charts → stalled lift detection → PR identification.
**Best for:** Users tracking progress over time, checking if they're improving.

### 4. "Help me break through a training plateau"
**Purpose:** Problem-specific entry. Targets users feeling stuck.
**Triggers:** Report parsing → identify exercises with flat/negative trends → analyze volume and intensity patterns → suggest plateau-breaking strategies (deload, rep range variation, exercise substitution, volume adjustment).
**Best for:** Intermediate/advanced users who feel they've stalled.
